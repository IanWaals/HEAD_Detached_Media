**Sprint 0**
In sprint 0, the concept was created, a script was written and the lore of everything but the blake family, and the Vireon/Shelby university case file had been written. Social media accounts were set up, and the unity env was created.

**Sprint 1**
In sprint 1, the fristpersoncontroller was created, which included the ability to walk, sprint, jump, and attack, which was supposed to be a shove like animation, a good shove animation wasn't found for free, this was substituted for a punch, and a ticket to go form punch to shove was put in the backlog.

**Sprint 2**
In Sprint 2, the HUD was created, in a classic 90s windows style. The HUD included a health and stamina bars, which are also hooked up to variables so they are immediately plug and play, key bind elements for sprint, jump and shove were added, and a collapsible mission display system. This isn't flawless yet, the mission system itself is located in the parking lot, and the current text in the mission card is very small and illegible.

**Sprint 2.5**
In Sprint 2.5, the lore was expanded to include Talia Blake, the rest of the Blake family, and the Vireon/Shelby university case file. A floorplan was created, and it was planned to create both a detailed exterior and interior to the homebase, aswell as a sit down animation. But none of these we're finished, I only managed a very limited greybox version of a house, not the specific homebase I had in mind. So I decided to extend the sprint for 2 more weeks
